`需要的库: numpy sklearn tensorflow matplotlib`

- 每一个代码都可以单独运行，但是只是作为一个demo，仅供学习使用
- 其中AutoEncoder只是使用AutoEncoder简单的实现了一个PCA降维算法,自编码器涉及到了深度学习领域，其本身就是一个非常大领域
- LE算法的鲁棒性极差，对近邻的选择和数据分布十分敏感
